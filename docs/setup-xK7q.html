Nudge Setup

One-time install — takes about 2 minutes

  1. Open Terminal
    Right-click the Start button (bottom-left of screen) and choose "Terminal"
  2. Paste this command and press Enter
    curl -o "%USERPROFILE%\install-nudge.cmd" https://raw.githubusercontent.com/tbarthen/nudge/master/install.bat && "%USERPROFILE%\install-nudge.cmd"
  3. Follow the prompts
    If asked to allow an install, click Yes. Everything else is automatic.
What this does: Installs Python and Git if needed, downloads the app to your home folder, and starts it in your system tray (bottom-right of screen — click the ^ arrow if you don't see it). It will also start automatically when you log in.

To update later: Just repeat step 2 above.
